Formily Request 是一个灵活、轻量、无侵入性，扩展 json schema 配置的请求数据插件。

- 组件无关，不用写额外的自定义组件
- 框架无关，不用关注是 Vue 还是 React
- 组件库无关，不用关注是 Ant Design 还是 Element Plus
- 高扩展性，可用于 Select、Radio、Checkbox、Cascader 等任何动态数据的组件
